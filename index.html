Welcome

This page provides access to Civil Air Patrol brand tools designed to help members create compliant visual materials using approved marks, formats, and naming conventions.

Before using these tools

  • Use official Civil Air Patrol names, titles, and organizational references.
  • Only use approved marks, emblems, and supporting graphics.
  • Review generated output for compliance before downloading or publishing.
  • When in doubt, contact the brand team for clarification.

Brand Tool Guidance

Use the appropriate tool for the type of branded asset you need to create.

Use the Subordinate Entity Graphic Generator for:

Directorates, programs, campaigns, and subordinate entity graphics that use the CAP wordmark with supporting subordinate text and optional approved secondary graphics.

Use the Email Signature Block Generator for:

Standardized email signatures for adult members, cadets, and NHQ staff using approved formatting and live preview.

Use the Emblem Generator for:

Heraldic and emblematic graphics that require approved CAP emblem formatting and presentation.
